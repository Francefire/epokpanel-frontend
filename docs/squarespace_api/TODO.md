Voici les documentations de l'API "Products" pertinentes pour construire un panel de gestion de produits, formatées en Markdown.

## Gestion des Produits (Products API)

* **Vue d'ensemble (Compréhension des attributs) :**
    * `https://developers.squarespace.com/commerce-apis/products-overview`
* **Récupérer toutes les pages de magasin (Nécessaire pour le `storePageId`) :**
    * `https://developers.squarespace.com/commerce-apis/retrieve-all-store-pages`
* **Récupérer tous les produits (Nécessaire pour le "bulk edit") :**
    * `https://developers.squarespace.com/commerce-apis/retrieve-all-products`
* **Récupérer des produits spécifiques :**
    * `https://developers.squarespace.com/commerce-apis/retrieve-specific-products`
* **Créer un produit (Utile pour les "template products") :**
    * `https://developers.squarespace.com/commerce-apis/create-product`
* **Mettre à jour un produit (Nécessaire pour le "bulk edit" et "highlighted products") :**
    * `https://developers.squarespace.com/commerce-apis/update-product`
* **Supprimer un produit :**
    * `https://developers.squarespace.com/commerce-apis/delete-product`

## Gestion des Images (Product Images API)

* **Uploader une image de produit :**
    * `https://developers.squarespace.com/commerce-apis/upload-product-image`
* **Assigner une image à une variante :**
    * `https://developers.squarespace.com/commerce-apis/assign-product-image-to-variant`
* **Réorganiser les images d'un produit :**
    * `https://developers.squarespace.com/commerce-apis/reorder-product-image`
* **Supprimer une image de produit :**
